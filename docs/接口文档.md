# 推荐系统接口文档

## 端口说明

| 服务 | 端口 | 说明 |
|------|------|------|
| recommend | 6058 | 数据元/限定词推荐服务 |
| tablemap | 6061 | 表映射/字段映射服务 |
| admin | 6070 | 向量库管理服务 |

---

## /autoexport/api 接口（端口 6058）

### 1. 数据元推荐

**接口地址**: `POST /autoexport/api/recommend`

**请求参数**:

```json
{
  "fieldsInfo": [
    {
      "cname": "姓名",
      "ename": "xm",
      "type": "string",
      "length": 50
    }
  ]
}
```

**测试命令**:

```bash
curl -X POST http://10.100.13.66:6058/autoexport/api/recommend \
  -H "Content-Type: application/json" \
  -d '{"fieldsInfo":[{"cname":"姓名","ename":"xm","type":"string","length":50}]}'
```

---

### 2. 数据元推荐（带扩展）

**接口地址**: `POST /autoexport/api/recommendWithExtend`

**请求参数**:

```json
{
  "lybEname": "acceptance",
  "fieldsInfo": [
    {
      "cname": "性别代码",
      "ename": "XBDM",
      "type": "string",
      "length": 1
    }
  ]
}
```

**测试命令**:

```bash
curl -X POST http://10.100.13.66:6058/autoexport/api/recommendWithExtend \
  -H "Content-Type: application/json" \
  -d '{"lybEname":"acceptance","fieldsInfo":[{"cname":"性别代码","ename":"XBDM","type":"string","length":1}]}'
```

---

### 3. RUN 通用字段模板

**接口地址**: `POST /autoexport/api/commonFields`

**请求参数**:

```json
{
  "primaryKeyFields": ["id", "name"],
  "RUN_SJLYXTFLDM": "01",
  "RUN_CJD_XZQHDM": "110101",
  "RUN_SJJLMGJB": "99",
  "RUN_GLHT_JYQK": "const('test')",
  "RUN_HSXX_JYQK": "const('')",
  "RUN_LYZYBM": "LY001"
}
```

**测试命令**:

```bash
curl -X POST http://10.100.13.66:6058/autoexport/api/commonFields \
  -H "Content-Type: application/json" \
  -d '{"primaryKeyFields":["id","name"],"RUN_SJLYXTFLDM":"01","RUN_CJD_XZQHDM":"110101"}'
```

### 9. 向量库重建

**接口地址**: `POST /autoexport/api/vectorIndex/rebuild`

**请求参数**:

```json
{
  "collections": ["all"]
}
```

或指定集合：

```json
{
  "collections": ["elements", "qualifiers", "dict_items"]
}
```

**测试命令**:

```bash
curl -X POST http://10.100.13.66:6058/autoexport/api/vectorIndex/rebuild \
  -H "Content-Type: application/json" \
  -d '{"collections":["all"]}'
```

## /autoexport/api 接口（端口 6061）

### 10. 表映射

**接口地址**: `POST /autoexport/api/tableMap`

**请求参数**:

```json
{
  "dbkCname": "人员信息表",
  "dbkFields": ["姓名", "性别", "出生日期", "联系电话"],
  "fLimitScore": 0.6,
  "tLimitScore": 0.7
}
```

```bash
curl -X POST http://10.100.13.66:6061/autoexport/api/tableMap \
  -H "Content-Type: application/json" \
```
